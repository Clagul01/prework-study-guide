Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The ul element creates an unordered list.
  • The img element requires a src attribute with reference to the location of the image file, and a alt attribute to display descriptive text when the image is not available.

CSS

  • A margin indicates how much space we want around an element.
  • A padding indicates how much space we want around the content of an element.
  • Class attribute allows us to share a CSS rule with any element of the same class.
  • The color property allows us to change the colour of text in an element.
  • We can use * attribute to assign the same css rules for every element on a webpage.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git clone repo-address: clones the repository into the IDE.
  • git add -A: adds a change in the working directory to the stage area.
  • git pull origin main: pulls changes from the origin branch, merging them to the locally checked-out branch.
